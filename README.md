# 5-Practical-Prompts-for-Class-Demonstrations
This repository contains 5 practical prompts designed for class demonstration on creating a professional recruitment poster for hiring Data Analysts. 

The prompts showcase how AI can be used to generate corporate-style designs with structured content, modern visuals, and effective call-to-action elements. This project highlights the application of prompt engineering in real-world business use cases such as recruitment marketing.

## Practical Prompts

### 1. Basic Poster Generation
Create a professional hiring poster for a company recruiting Data Analysts. Include a modern office background, diverse team collaboration, and bold headline ‘Join Our Team’ with a clean corporate design.

### 2. Enhanced Visual Design
Design a corporate recruitment poster for Data Analysts using a blue and white color scheme. Add data dashboards, charts, and graphs in the background, with a diverse team working in a modern office. Include ‘We’re Hiring Data Analysts’ and ‘Apply Now’ CTA.

### 3. Content + Structure
Create a hiring poster for Data Analysts with sections: What You’ll Do, Skills Required, and Why Join Us. Include SQL, Excel, and data visualization skills.

### 4. Advanced Marketing Prompt
Design a high-conversion recruitment poster for Data Analysts targeting young professionals with strong CTA and modern UI elements.

### 5. Social Media Version
Create an Instagram-friendly recruitment poster with minimal text and bold ‘Join Our Team’ headline.
